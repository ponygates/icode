本次更新

  • 更漂亮的字标：宽终端用 ansi_shadow 实心块字标（36 列宽），中等终端用 slant 斜体字标（27 列宽），并施加 洋红→紫→蓝→青 逐行渐变着色，比旧有的方块字标更精致。
  • Claude Code 式方框：Logo 下方的 模型 / Provider / 工作目录 / 提示 全部收进 ASCII 方框，左右用 │ 竖墙隔开；Model 与 Provider 在同一行左右分隔（放不下时自动换行）。
  • 窗口自适应：watchResize() 每 150ms 轮询终端尺寸，缩放窗口即时重排，中文按宽度 2 折行，不再错位。
  • 可关闭 + 文案：Esc / 回车 / 输入首字符即关闭；/welcome 重新打开；底部提示语已中/繁/英本地化。
  • 窄屏优雅降级：<40 列不显示方框（纯缩进信息），<30 列不显示字标（仅单行文字），始终可读。
